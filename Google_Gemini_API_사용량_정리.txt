# Google Gemini API 무료 사용량 정리

## 📊 무료 플랜 (Free Tier) 제한

### 요청 제한
- **분당 요청 수 (RPM)**: 15개
- **일일 요청 수 (RPD)**: 1,500개
- **월간 제한**: 없음 (일일 제한만 준수하면 됨)

### AI Story Lab 기준 사용량
- **대본 1개 생성** = 요청 2개
  - 원본 대본 분석: 1개 요청
  - 새로운 대본 생성: 1개 요청

### 실제 가능한 사용량
- **하루 최대**: 약 750개 대본 생성 가능
- **분당 최대**: 약 7~8개 대본 생성 가능
- **일반 사용**: 충분한 양

## ⚠️ 제한 초과 시 대응

### 분당 제한 초과 (15개 초과)
- 일시적으로 API 차단됨
- 1분 대기 후 자동 복구
- 에러 메시지: "Rate limit exceeded"

### 일일 제한 초과 (1,500개 초과)
- 다음 날 자정(UTC)까지 차단
- 유료 플랜으로 업그레이드 필요
- 에러 메시지: "Quota exceeded"

## 🔐 API 키 보안 관리

### 현재 적용된 보안 조치
✅ .env 파일은 로컬에만 존재
✅ GitHub에 업로드되지 않음 (.gitignore 설정)
✅ Vercel은 환경변수로 안전하게 관리
✅ API 키 노출 방지 완료

### API 키 관리 주의사항
❌ GitHub, 공개 저장소에 업로드 금지
❌ 채팅, 이메일 등으로 공유 금지
❌ 스크린샷에 키가 보이지 않도록 주의
✅ 유출 시 즉시 키 삭제 및 재발급

## 📈 사용량 모니터링

### 확인 방법
1. https://aistudio.google.com/app/apikey 접속
2. API 키 목록에서 사용량 확인
3. 일일/월간 사용량 통계 확인 가능

### 모니터링 권장 사항
- 주기적으로 사용량 확인
- 제한에 가까워지면 사용 조절
- 필요시 유료 플랜 고려

## 💰 유료 플랜 정보

### Pay-as-you-go (종량제)
- 요청당 과금
- 제한 없음
- 사용한 만큼만 지불
- 자세한 가격: https://ai.google.dev/pricing

### 언제 업그레이드를 고려해야 하나?
- 하루 750개 이상의 대본 필요
- 동시 다수 사용자 서비스
- 비즈니스/상업적 용도

## 🎯 최적 사용 팁

1. **일반 사용자**: 무료 플랜으로 충분
2. **테스트/개발**: 무료 플랜 추천
3. **대량 생성 필요**: 유료 플랜 고려
4. **사용량 분산**: 시간대별로 분산 사용
